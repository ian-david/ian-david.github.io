Hello There

I am David Okpare

Ever since I was a kid, I absolutely loved computers. As I grew older my passion for computers translated into programming. I use my eye for design and my obsessive attention to detail to create pixel-perfect interfaces. More than anything I love being creative and get excited about being able to create products that will visually stun people.

What I Do

These are some of the things I do on and off line

Web Design

I enjoy creating aesthetically beautiful and functional user interfaces, at the same time not ignoring the experience the users get from the page. I create web pages using both codes and tools such as HTML5, CSS3, Javascript, jQuery, Bootstrap, Bootstrap Studio and WordPress.

Illustration

When I'm not coding, I make designs for print and web works with images, lines, typography, textures, color, lighting contrasts, and shapes. I use tools such as Adobe's Photoshop, Illustrator and Muse and Corel's Graphic Suites.

Open Source Contributing

Contributing to open source can be a rewarding way to learn, teach, and build experience in just about any skill you can imagine. I often create repositories on GitHub and make my projects public and try to report issues or pull requests on other repositories.

Software Development

I don't limit my self to just the web, but like to explore other possibilities by building softwares. I have built softwares with C, C++ and Java.
